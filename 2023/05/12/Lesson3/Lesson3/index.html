3. Python as a Calculator

Objective: By the end of the lesson, students will be able to perform basic mathematical operations using Python as a calculator.

Audience: Beginner-level students who have no prior experience with Python.

Duration: 30-45 minutes

Materials:
• A computer with Python installed
• Presentation slides or a whiteboard to display content
• Handout with practice exercises

Lesson Plan:

**Introduction (5 minutes)**•

  • Introduce the concept of using Python as a calculator

• Explain the benefits of using Python instead of a traditional calculator, such as the ability to store results and use variables in calculations

Basic Math Operations (10 minutes)

• Explain the basic mathematical operations that can be performed using Python: addition, subtraction, multiplication, and division
• Demonstrate how to perform each operation using the Python interpreter or a code editor

• Use the following examples:

Order of Operations (10 minutes)

• Explain the order of operations in Python and how it can affect the outcome of a calculation

• Demonstrate how to use parentheses to group operations and override the default order of operations

• Use the following examples:

Variables and Assignment (10 minutes)

• Introduce the concept of variables and how they can be used in Python
• Explain how to assign values to variables using the = operator
• Use the following examples:

Conclusion (5 minutes)

• Review the key concepts covered in the lesson
• Provide students with a handout or link to additional resources for practice
• Encourage students to continue exploring Python as a calculator and experimenting with different math operations and variables.

Handout/exercises for students

Complete the following exercises to practice using Python as a calculator. Type your answers directly into the Python interpreter or a code editor and press enter to see the output.

  1. Calculate the sum of 8 and 12.
  2. Calculate the difference between 15 and 6.
  3. Calculate the product of 5 and 9.
  4. Divide 20 by 4.
  5. Calculate the value of the expression (6 + 2) * 3.6. Calculate the value of the expression 4 + 3 * 5.
  6. Calculate the value of the expression (4 + 3) * 5.
  7. Assign the value 10 to a variable called x, then calculate x + 7.
  8. Assign the value 6 to a variable called y, then calculate x - y, where x is the variable from the previous exercise.
  9. Calculate the value of the expression (x + y) * 2, where x and y are the variables from the previous two exercises.

Challenge Exercises:

These exercises are more challenging and may require some additional research. Don’t worry if you can’t solve them right away!

  1. Calculate the square root of 64 using Python.
  2. Calculate the value of pi (π) to 5 decimal places using Python.
  3. Use Python to calculate the area of a circle with radius 5. (Hint: you’ll need to use the value of pi in your calculation.)
  4. Use Python to calculate the volume of a sphere with radius 3.
    (Hint: you’ll need to use the value of pi in your calculation.)
  5. Use Python to solve the quadratic equation 2x^2 + 5x - 3 = 0.
    (Hint: you’ll need to use the sqrt function from the math module.)
